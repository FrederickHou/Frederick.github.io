LeetCode 另一个树的子树
发表于 2020-05-08| 更新于 2020-06-05|技术笔记数据结构与算法笔记
字数总计:580|阅读时长: 2 分钟
| 阅读量:|评论数:

题目

给定两个非空二叉树 s 和 t，检验 s 中是否包含和 t 具有相同结构和节点值的子树。s 的一个子树包括 s 的一个节点和这个节点的所有子孙。s 也可以看做它自身的一棵子树。

示例 1:
给定的树 s:

    3
   / \
  4   5
 / \
1   2

给定的树 t：

  4 
 / \
1   2

返回 true，因为 t 与 s 的一个子树拥有相同的结构和节点值。

示例 2:
给定的树 s：

    3
   / \
  4   5
 / \
1   2
    /
   0

给定的树 t：

  4
 / \
1   2

返回 false。

解题方法

递归检测

思路

挨着遍历树的节点，判断当前节点对应的树，和给定的树是否相同，相同则返回true。不相等继续遍历下一节点。

解题示例

性能：

  • 耗时：24ms
  • 内存消耗：6.2M

树节点的前序遍历

思路

判断该树是否是给定树的t子树，只需要对两个树进行先序遍历得到一个序列，然后判断给定的t子树得到的序列是不是目标s树序列的子序列即可。
如题示例2进行先序遍历:
s树：3、4、1、null、null、2、0、null、5、null、null
t树：4、1、null、null、2、null、null
通过对比t树得到的序列显然不是s树得到序列的子序列。

解题示例

/**
* Definition for a binary tree node.
* type TreeNode struct {
*     Val int
*     Left *TreeNode
*     Right *TreeNode
* }
*/

func isSubtree(s *TreeNode, t *TreeNode) bool {
    var resS []string
    var resT []string
    preorderTraversa(s,&resS)
    preorderTraversa(t,&resT)
    a,_ := json.Marshal(resS)
    b,_ := json.Marshal(resT)
    return strings.Contains(strings.Trim(string(a),"[]"),strings.Trim(string(b),"[]"))
}

func preorderTraversa(s *TreeNode, res *[]string){
    if s == nil{
        *res = append(*res,"null")
        return
    }
    *res = append(*res,strconv.Itoa(s.Val))
    preorderTraversa(s.Left,res)
    preorderTraversa(s.Right,res)
}

性能：

  • 耗时：20ms
  • 内存消耗：6.4M
