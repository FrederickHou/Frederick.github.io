堆基础介绍

  • 1.完全二叉树，双亲节点和孩子节点的关系

    array[0，…，n-1]表示完全二叉树数组顺序存储
    1.节点array[i] 的父节点为 i==0 ? null : array[i-1/2]
    2.节点array[i]的左孩子为，array[2i + 1]
    3.节点array[i]的右孩子为，array[2    i + 2]

  • 2.堆在数组中的表示方式

    堆的逻辑结构是一颗完全二叉树，存储结构是一个数组。
    数组为完全二叉树的层次遍历的结果。

  • 3.大根堆和小根堆的定义

    前提0 <= i < (n-1)/2
    1.满足array[i] <= array[2i + 1] && array[i] <= array[2i + 2] 为小根堆。
    2.满足array[i] >= array[2i + 1] && array[i] >= array[2i + 2] 为大根堆

小根堆

创建小根堆

对于叶子节点，不用调整次序，根据满二叉树的性质，叶子节点比内部节点的个数多1.所以i=n/2 -1 ，不用从n开始。就是从最后一个非叶子结点的结点开始，自下往上调整堆 。

小根堆插入元素

插入一个元素：新元素被加入到heap的末尾，然后更新树以恢复堆的次序。每次插入都是将新数据放在数组最后,(在该新节点插入之前，可以发现从这个新数据的父结点到根结点必然为一个有序的数列。)然后从最后一个非叶节点（i=n/2-1）开始,自下往上进行堆的顺序调整 。如下图所示：

删除小根堆堆的最小元素

按定义，堆中每次都删除第0个数据。为了便于重建堆，实际的操作是将最后一个数据的值赋给根结点，堆的元素个数-1，然后再从根结点开始进行一次自上向下的调整 。

堆排序

堆排序算法先建立堆(大根堆或者小根堆)，然后通过堆与末尾元素互换，调整堆，重复这个过程，最终完成排序。

  • 小根堆 ：堆的第一个元素即最小元素，将其插入列表中，堆所在的数组长度-1，将堆数组最后一个元素赋值给堆数组的第1个元素，重新调整堆。然后继续重复此操作，得便可得到从小到大排序序列。
  • 大根堆 ： 堆的第一个元素即最小元素，将其插入列表的末尾(后续依次从后往前插入)，堆所在的数组长度-1，将堆数组最后一个元素赋值给堆数组的第1个元素，重新调整堆。然后继续重复此操作，得便可得到从小到大排序序列。
  • 时间复杂度 ： 堆排序的时间复杂度为O(nlgn)。

例题

示例1(简单)

设计一个找到数据流中第K大元素的类（class）。注意是排序后的第K大元素，不是第K个不同的元素。

你的 KthLargest 类需要一个同时接收整数 k 和整数数组nums 的构造器，它包含数据流中的初始元素。每次调用 KthLargest.add，返回当前数据流中第K大的元素。

示例:

int k = 3;
int[] arr = [4,5,8,2];
KthLargest kthLargest = new KthLargest(3, arr);
kthLargest.add(3);   // returns 4
kthLargest.add(5);   // returns 5
kthLargest.add(10);  // returns 5
kthLargest.add(9);   // returns 8
kthLargest.add(4);   // returns 8

思路

数据流中第K大元素即可维护一个长度为K的数组，并对其建立小根堆。小根堆的对顶元素即是第K大的元素。
