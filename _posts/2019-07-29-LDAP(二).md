# Docker安装LDAP

    安装LDAP分server端和客户端两部分的安装部署
---

## Server端OpenLDAP安装

**拉取镜像**

    docker pull osixia/openldap:1.2.4

**运行镜像**

    docker run -p 389:389 -p 636:636 
        --name my-openldap-container 
        --detach osixia/openldap:1.2.4
